Heart Failure ML Project

The project focuses on heart failure as a binary classification task (1 for death, 0 for survival), utilizing Kaggle data, exploratory techniques, and machine learning algorithms, and concludes with visualizing and comparing their accuracies through a bar chart.

Alcohol Analysis Clustering Project

The project uses an Italian wine dataset with 13 chemical components to apply k-Means Clustering, aiming to categorize wine samples based on similar malic acid and alcohol content.

Antarctic Penguins Analysis Regression Project

The project applies linear regression and KNN to a penguin dataset for classifying species by bill depth and predicting body mass from flipper length, featuring real-world ML examples with Python visualization..
